# CaffemochaY's AviUtl Scripts

自作のAviUtlスクリプトたちです

## ダウンロード/インストール方法

右上の`Code`の`Download ZIP`からDLして、`script`フォルダの中身をスクリプトが機能する場所に置いてください

## 各種スクリプトの説明

スクリプトファイルの先頭にコメントで説明文が記載してあります  
markdownみたいな書き方をしているので、コピペしてmarkdownにして見るといいかもしれません

## リスト

### ソフトフォーカス

- `ソフトフォーカス.anm`

  - 画像(立ち絵、背景など)を柔らかい雰囲気にする
  - ついでに立体っぽさを演出出来るかも
